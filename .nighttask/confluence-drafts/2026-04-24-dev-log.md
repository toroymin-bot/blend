## 🤖 Phase 3.7d — AI-generated Model Descriptions

**Commit**: `3427ce8`

### 핵심 아이디어
새 모델 출시 → 사람이 META_OVERRIDES 수동 작성하는 병목 제거. **Gemini 2.5 Flash가 자동으로 displayName, description_ko, description_en, tier 생성** + 검증 레이어로 품질 가드.

### 검증 레이어 (Jobs식 품질 통제)
- description_ko: 15자 이내, 해요/예요/어요 종결, 금지어(플래그십, SOTA, 최첨단) 차단
- description_en: 40자 이내, 마케팅 단어 금지(flagship, cutting-edge, revolutionary)
- displayName: 대문자 시작 강제, 날짜 스탬프 차단
- 검증 실패 시 3회 재시도, 그래도 실패면 heuristic fallback

### 자동 영속화
- `persistNewMetas()` 함수: AI 생성 메타를 `scripts/update-models.ts` 자체에 append
- `// ===== AUTO-APPEND BELOW =====` 마커 기준
- 다음 cron 실행부터 정적 lookup → AI 호출 비용 0

### 핵심 발견
- Gemini 2.5 Flash는 **`thinkingBudget: 0` + `maxOutputTokens: 2000`** 필수. 기본값이면 thinking token이 출력 budget 잡아먹어 응답이 `{`만 나옴
- 검증 정규식 `[해예어][요]$` → `요$|다$` 완화 필요 (워요/에요/이에요 등 수용)

### 변경 파일
- `scripts/generate-descriptions.ts` (신규) — Gemini 호출 + validate + retry
- `scripts/update-models.ts` — generateMeta 통합 + persistNewMetas
- `.github/workflows/update-models.yml` — diff/commit에 update-models.ts 포함

## ⌘K Phase 3.8 — History Overlay + Export Dropdown

**Commit**: `8b016d8`

### 문제
design1은 self-contained 로컬 state만 사용 → **새로고침하면 대화 사라짐**. 히스토리/내보내기 UI도 없음.

### 해결 — 5개 파일 통합 작업

**1. `src/stores/d1-chat-store.ts` (신규)**: design1 전용 Zustand 스토어. localStorage 키 `d1:chats`. 메인 useChatStore 미사용 (수십 개 액션이라 design1 minimum subset만 재구현).

**2. `src/modules/chat/history-overlay-design1.tsx` (신규)**: 스포트라이트 검색 + 4필터 탭(오늘/이번주/이번달/전체) + 날짜 그룹(오늘/어제/이번주/이번달/older) + 방향키 네비 + ESC 닫기 + 호버 삭제.

**3. `src/modules/chat/export-dropdown-design1.tsx` (신규)**: md/txt/json/pdf 4개 포맷. 외부 클릭/ESC로 닫힘.

**4. `src/modules/chat/export-utils-design1.ts` (신규)**: D1Chat → 기존 Chat 타입 adapter. 기존 `export-chat.ts` 재활용 (downloadChat, downloadChatAsJSON, downloadChatAsPDF).

**5. `src/modules/chat/chat-view-design1.tsx` (수정)**:
- messages 변화마다 자동 저장 (chat ID + createdAt 안정 유지)
- 첫 user 메시지로부터 title 유도
- ⌘K (Mac) / Ctrl+K (Windows) 전역 단축키
- 히스토리 아이콘 → 오버레이 / Share 아이콘 → 내보내기 드롭다운
- messages.length === 0일 때 내보내기 비활성 (opacity 0.35, not-allowed, 설명 툴팁)
- 히스토리에서 클릭 시 messages + model + createdAt 복원

### UX 디자인 결정
- 히스토리 max-width 720px, max-height 70vh, fade+rise 애니메이션
- 결과 없음 빈 상태 분기: 검색결과 없음 vs 저장된 대화 자체 없음
- 내보내기 dropdown 4개 항목 각자 아이콘 + 부제목(.md/.txt/백업용/인쇄)
